# data prepare
要实现行人检测与路面缺陷检测，需要对这两类样本进行标注。  
本项目采用COCO数据集提取行人样本，RDD2022数据集提取路面缺陷样本。  
最终处理得到65535个行人数据集与38385个路面缺陷数据集。
